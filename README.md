# ProFormatInvoice

Dolibarr module for generating proforma invoices from sales orders or proposals.

## Features

- Generate proforma invoices from sales orders
- Generate proforma invoices from commercial proposals
- Custom proforma invoice templates
- Multi-language support
- PDF export
- Email sending capabilities

## Installation

1. Copy the module folder to `custom/` directory in your Dolibarr installation
2. Go to **Home → Setup → Modules**
3. Enable the **ProFormatInvoice** module
4. Configure the module settings

## Requirements

- Dolibarr 14.0+
- PHP 7.4+
